# 🔍 大霄式价值投资探测器

践行"余钱投资、理性投资、价值投资"，自动探测"钻石底"基因，远离"黑五类"。

## 功能特性

- 📊 **牛熊判断**：基于沪深300估值分位数，自动识别市场周期
- 🎯 **股票雷达**：五维评分（价值/质量/安全/成长/动量）
- 🚨 **预警系统**：抄底/减仓/清仓/成长观察四重信号
- 📈 **业绩探测**：ROE/增速/分红持续性分析
- 💼 **组合建议**：核心仓+卫星仓+现金动态配置

## 快速开始

### 1. Fork本仓库

点击右上角Fork按钮，复制到自己的GitHub账号。

### 2. 配置GitHub Action

进入Settings → Secrets and variables → Actions，添加：

- `GITHUB_TOKEN`：自动存在，无需添加
- `TCB_WEBHOOK`（可选）：腾讯云触发地址

### 3. 手动触发首次运行

进入Actions → Update Data → Run workflow，等待5-10分钟。

### 4. 查看数据

数据生成在`data/`目录：
- `market/index_valuation.json`：市场状态
- `stocks/YYYY-MM-DD.json`：全市场个股数据
- `indicators/warning_signals.json`：预警信号

### 5. 部署前端（可选）

Vue3前端在`web/`目录，可部署到：
- 腾讯云静态托管（推荐）
- Vercel/Netlify
- GitHub Pages

## 数据更新频率
